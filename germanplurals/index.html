German Plurals is a fun German plural noun quiz game in which the player tries to guess the correct plural form for a set of nouns.

Presently the game is in development:

  • Contains 350 nouns adhering to the Goethe-Institut A1 examination, separated into 7 groups of 50.
  • Words are pronounced uses speech synthesis.
  • Each correct answer earns one point. A false answer resets the score to zero.
  • Progress and difficult nouns can be viewed from a list overview.
Development roadmap:
  • Add nouns adhering to Goethe-Institut A2 and B1.
  • Release test version on Android and iOS.
  • Official release on the App Store and Google Play.
NOTE:
  • Nouns which have a plural form that is not part of Goethe A1-B1 certifications are denoted "Seldom used", e.g. das Wasser/die Wasser, das Leben/die Leben etc. When questioned, these nouns have a ? button in the top right which displays a popup with relevant information.
  • Although Goethe-Institut covers German in Germany, Austria and Switzerland, only Hochdeutsch used in Deutschland is considered. Thus die E-Mail instead of das E-Mail, die Fahrkarte instead of das Billet, die Balkone as the plural of der Balkon etc.
  • German Plurals is a game in which the player tests their knowledge of the plural forms for a set of German nouns, it isn't designed to teach vocabulary. No English (or other language) translations for the German nouns will be included.
